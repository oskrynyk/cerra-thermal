# Thermal Stress and Climate Zones in Europe and Northern Africa

Human thermal comfort analysis for Europe and Northern Africa using high-resolution climate data (CERRA, ERA5, E-OBS).

## Thermal Indices

- **MRT** - Mean Radiant Temperature
- **UTCI** - Universal Thermal Climate Index  
- **NET** - Normal Effective Temperature

## Data Sources

- **CERRA** (5-10 km) - High-resolution European reanalysis
- **ERA5** (31 km) - Global atmospheric reanalysis
- **E-OBS** (11 km) - European observational dataset

## Usage

```bash
pip install -r requirements.txt
python scripts/main.py
```

Most results saved to `outputs/` folder.
